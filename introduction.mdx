---

ewelink-api is a JavaScript library that lets you interact with the eWeLink API using your regular account credentials. It works in Node.js scripts, browser environments, and serverless functions, giving you programmatic control over Sonoff and other eWeLink-compatible smart home devices — including power state, real-time event listeners, power consumption, and offline LAN control via Zeroconf.

## Installation

```bash
npm install ewelink-api
```

## Key features

- **Cross-environment** — runs in Node.js, browsers, and serverless runtimes
- **Device control** — turn devices on, off, or toggle power state
- **Power monitoring** — read power consumption on devices like Sonoff POW
- **Real-time events** — listen for device state changes over WebSockets
- **LAN mode (Zeroconf)** — control devices on the local network without an internet connection
