---
title: "Get started with ewelink-api in under 5 minutes"
description: "Install ewelink-api, find your account region, and send your first power command to an eWeLink device from JavaScript in minutes."
---

ewelink-api connects to the eWeLink cloud using your account credentials. The steps below take you from a fresh install to toggling a real device — all you need is your eWeLink email and password.

<Steps>
  <Step title="Install the package">
    Add ewelink-api to your project using your preferred package manager.

    <CodeGroup>
      ```bash npm
      npm install ewelink-api
      ```

      ```bash yarn
      yarn add ewelink-api
      ```

      ```bash pnpm
      pnpm add ewelink-api
      ```
    </CodeGroup>
  </Step>

  <Step title="Find your region">
    eWeLink accounts are assigned to a region. You must pass the correct region when instantiating the library or the API returns a 400/401 error.

    Use `getRegion` to look up your region from your email and password. This method works only when the class is initialized with credentials.

    ```javascript
    const ewelink = require('ewelink-api');

    const connection = new ewelink({
      email: '<your ewelink email>',
      password: '<your ewelink password>',
      region: 'us',
    });

    const region = await connection.getRegion();
    console.log(region);
    ```

    Response:

    ```javascript
    {
      email: 'user@email.com',
      region: 'us'
    }
    ```

    <Note>
      The default region is `us`. If your account is in a different region, pass the correct value in the `region` field when constructing the class. Passing the wrong region will cause 400 or 401 errors on every request.
    </Note>
  </Step>

  <Step title="Instantiate the class">
    Create a connection using your email, password, and the region you confirmed in the previous step.

    ```javascript
    const ewelink = require('ewelink-api');

    const connection = new ewelink({
      email: '<your ewelink email>',
      password: '<your ewelink password>',
      region: '<your ewelink region>',
    });
    ```
  </Step>

  <Step title="Get all devices">
    Call `getDevices` to retrieve the list of devices associated with your account.

    ```javascript
    const devices = await connection.getDevices();
    console.log(devices);
    ```

    The response is an array of device objects. Each object includes the device ID you need to control a specific device.
  </Step>

  <Step title="Control a device">
    Use `setDevicePowerState` to turn a device on or off. Pass the device ID from the previous step and the target state: `on`, `off`, or `toggle`.

    ```javascript
    const status = await connection.setDevicePowerState('<your device id>', 'on');
    console.log(status);
    ```

    Response:

    ```javascript
    {
      status: 'ok',
      state: 'on',
      channel: 1
    }
    ```

    For multi-channel devices like the Sonoff 4CH, pass the channel number as a third argument:

    ```javascript
    const status = await connection.setDevicePowerState('<your device id>', 'on', 2);
    console.log(status);
    ```
  </Step>
</Steps>
